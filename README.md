# 🧠 CAPTCHA Solver

A powerful and flexible CAPTCHA solving toolkit designed for automation workflows, scraping systems, and bot integrations.

---

## 🚀 Features

- ✅ Multi-provider support  
- ⚡ Fast solving (async supported)  
- 🔁 Callback-based result handling  
- 🔌 Easy API integration  
- 🧩 Modular architecture  
- 🌐 Proxy support (IP:Port:User:Pass)  
- 🔐 Session-based solving  

---

## 🧩 Supported CAPTCHA

- reCAPTCHA v2  
- AWS WAF  
- hCaptcha  
- GeeTest  
- MTCaptcha  
- Prosopo  
- TikTok  
- Binance  
- Tencent  
- Slide CAPTCHA
