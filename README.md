🧠 Captcha Solver

A powerful and flexible CAPTCHA solving toolkit designed for automation workflows, scraping systems, and bot integrations. Supports multiple CAPTCHA providers with an easy-to-use interface and callback system.

🚀 Features
✅ Multi-provider support
reCAPTCHA v2
AWS WAF
hCaptcha
GeeTest
MTCaptcha
Prosopo
TikTok
Binance
Tencent
Slide Captcha
⚡ Fast solving with async support
🔁 Callback-based result handling
🔌 Easy API integration
🧩 Modular design for extending new CAPTCHA types
🛠 Proxy support (IP:Port:User)
🔒 Session-based solving
